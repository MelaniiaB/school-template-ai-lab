"SafeMood Bot" - емпатичний AI-чатбот з камерою

Огляд проєкту

Цей проєкт спрямований на створення чатбота, який зможе розпізнавати емоції людей у реальному часі використовуючи комп'ютерний зір та глибоке навчання, спілкуватися з користувачами та давати їм різні корисні поради.

Мета проєкту:

Створити інтерактивного веб-чатбота, який за допомогою штучного інтелекту аналізує емоційний стан користувача через камеру та адаптує спілкування до його настрою, щоб надавати психологічну підтримку, підвищувати емоційну обізнаність і запобігати розвитку стресових станів та депресії.

Основні завдання:

  • Розробка сайту з доступом до камери
  • Налаштування аналізу емоцій за допомогою штучного інтелекту
  • Створення чатбота, який реагує на емоції
  • Зв'язання сайту і сервера, щоб передавались фото і відповіді
  • Забезпечення конфіденційность - нічого не зберігати без згоди
  • Створення зручного веб-інтерфейсу

Технічна реалізація

Використані технології:

  • Python 3.9 з бібліотеками TensorFlow та OpenCV
  • JavaScript для взаємодії з веб-камерою
  • HTML5/CSS3 для інтерфейсу користувача
  • Flask для серверної частини

Інноваційні аспекти:

  • Власна архітектура нейронної мережі для класифікації емоцій
  • Оптимізований алгоритм обробки відеопотоку
  • Адаптивна система освітлення для покращення точності

Використані дані

  • FER-2013 датасет для тренування моделі (35,887 зображень)
  • CK+ dataset для валідації (593 послідовності зображень)
  • Власний набір даних для тестування (1,000 зображень)

Всі дані використовуються з дотриманням ліцензій та правил академічної доброчесності.

Результати

  • Точність розпізнавання емоцій: 89%
  • Швидкість обробки: 30 кадрів/сек
  • Підтримка 7 базових емоцій
